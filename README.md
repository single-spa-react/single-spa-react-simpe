## ⌨️ 本地开发

```bash
$ git clone git@github.com:single-spa-react/single-spa-react-simple.git
$ cd single-spa-react-simple
$ npm install
$ npm start
```

打开浏览器访问 http://127.0.0.1:8000/project1/。
